# DVE
Delphi voxel engine

'''Delphi Voxel Engine'''

<u>Uses</u>
* OpenGL 3.1+ through [https://github.com/SaschaWillems/dglOpenGL dglOpenGL]
* [https://github.com/neslib/FastMath Neslib.FastMath]
<u>Overview</u>
* Simple, only 3 units. (Plus dependencies)
* As prototype template
<u>Features</u>
* Frustrum culling
* Block-block culling
* Ambient occlusion
* Chunks
